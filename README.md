# ⚡ Teleporty

Physics-grounded teleportation research. No handwaving, no sci-fi assumptions — just actual science, math, and a honest look at what's possible.

## What's in here

- **research/** — notes on government docs (CIA STARGATE, DIA DIRD #18, AFRL 2004), state of quantum teleportation, and theoretical frameworks (wormholes, ER=EPR, warp drive)
- **research/papers/** — deep-dives on key papers: Bennett 1993, Alcubierre 1994, Maldacena-Susskind 2013, Ford-Roman 1999, and more
- **math/** — calculations and derivations (in progress)
- **architecture/** — design docs for a conceptual teleporter (in progress)
- **index.html** — project landing page with a little guy you can throw around

## The honest answer

Three things get called "teleportation." They're very different:

| Thing | Status | Barrier |
|---|---|---|
| Quantum state teleportation | **Demonstrated** (1,400 km, 2017) | Doesn't move matter — states only |
| Traversable wormholes | **Valid GR math** | ~Jupiter mass of exotic matter required |
| Scan + transmit + reconstruct | **Conceptually coherent** | 10²⁸ bits, decoherence, Heisenberg |
